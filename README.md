# AI 實戰手冊

14 篇從零開始的 AI 協作指南。不講理論，只講我實際在用的流程、工具和思維方式。

## 這是什麼

我不是工程師，也沒讀過任何 AI 書。這 14 篇是我從 2024 年開始用 AI 工作，一路摸索出來的方法。

每一篇都包含：
- 我原本的做法 vs 現在的做法
- 完整流程拆解
- 可直接複製的 Prompt
- 常見卡關點和解法
