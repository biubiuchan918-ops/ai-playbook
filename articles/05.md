# 05｜用 AI 做投資研究：從財報到決策的完整流程

## 這篇解決什麼問題

投資研究需要大量閱讀和整理，但大多數散戶沒有時間——不是不想研究，是一打開財報就淹沒在數字裡，最後因為資訊過載反而什麼決定都沒做。這篇示範如何把研究流程壓縮到 1 小時。

## 我原本的做法 → 現在的做法

**原本：** 花一整天讀財報、看分析師報告、刷投資討論區。讀完之後腦子一團漿糊，因為看多和看空的理由都有道理，最後還是不敢做決定。

**現在：** 用固定的三步驟 prompt 流程——10 分鐘快速掃描、30 分鐘深入分析、20 分鐘整理成投資備忘錄。AI 負責「收集和整理資訊」，我負責「做判斷」。兩件事分開做，清楚得多。

拿 RKLB（Rocket Lab）做例子。我對這家公司感興趣，但如果用傳統方式研究——讀完財報、看分析師報告、翻投資討論區——至少一整天。用這套三步驟流程，我一小時內就有了一份結構化的投資備忘錄，看多看空理由都列清楚，還附了「什麼條件下我應該行動」的觸發點。

## 完整流程

### 步驟一：快速掃描（10 分鐘）

```
請幫我做 [公司名稱] 的快速投資掃描：

1. 公司基本面：做什麼的、主要營收來源、競爭優勢
2. 最近 1-2 季的財務表現重點（營收、毛利率、EPS 趨勢）
3. 近期重大事件或新聞（3 個月內）
4. 市場對這家公司的主要爭議或疑慮

請用重點條列，每點不超過 2 句話。
這是初步掃描，我之後會深入研究。
```

### 步驟二：深入分析（30 分鐘）

```
基於剛才的掃描，我想深入了解 [你最關心的 2-3 個面向]。

請幫我分析：
1. [面向A] 的具體數據和趨勢
2. [面向B] 跟同業比較的情況
3. [面向C] 未來 1-2 年可能的發展

對於每個面向，請同時給我：
- 看多的理由（bull case）
- 看空的理由（bear case）
- 你建議我進一步查證的數據來源
```

這裡有一個關鍵：**永遠要求 AI 同時給你看多和看空的理由。** 如果你只問「這公司好不好」，AI 會傾向給你正面答案。強制它雙面論述，你才能看到完整的圖景。

### 步驟三：投資備忘錄（20 分鐘）

```
請把我們剛才的研究整理成一份「投資備忘錄」，格式如下：

## 一句話摘要
（這家公司值不值得進一步研究，為什麼）

## 三個看多理由
## 三個看空理由
## 我還需要確認的事情（checklist）
## 決策觸發點
（什麼情況發生時我應該買入/賣出/觀望）
```

最後那個「決策觸發點」是整份備忘錄最重要的部分。它讓你在研究階段就設好條件，不是等到市場波動的時候才臨時做決策。

## 常見卡關點

**AI 的財務數據可能不是最新的**
AI 不是彭博終端。數字永遠要用官方財報交叉驗證。AI 的價值在幫你快速整理和結構化資訊，不是給你準確的即時數據。

**容易被 AI 的分析說服**
AI 擅長把任何觀點講得很有道理。如果你覺得它的分析「完全正確」，那你可能掉進了確認偏誤——它只是在用很有說服力的方式呈現你已經相信的東西。這就是為什麼要強制雙面論述。

**研究了很多但不敢做決定**
這代表你的決策觸發點沒設好。「等更多資訊」是一個永遠到不了的終點。在研究階段就設好具體條件：「如果下季營收成長超過 X%，我就加碼」。條件到了就執行，不再猶豫。
